felx布局

  • flex布局是用来瓜分剩余宽度的
    • 用它来替代之前的div table 和float 等布局
    • 更加的强大
  • 存在兼容问题：
    • IE10以下是不支持的
    • Android4，1/4.2/4.3存在问题 苹果safari7存在问题
    • Webkit内核的浏览器，必须加上-webkit前缀

      如何瓜分剩余宽度

  • 首先剩余宽度是怎么算出来的？
    剩余宽度 = 父元素宽度 - 子元素宽度
  • 如何分配这个剩余宽度？
    1. 父元素开启display:flex;
      • 容器默认存在两根轴：水平的主轴（main axis）和垂直的交叉轴（cross axis）
    2. 子元素的分配规则：
      用flex 0/1/2/3占的份数
